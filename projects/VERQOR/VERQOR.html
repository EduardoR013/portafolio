Descripción del Proyecto

Este proyecto tuvo como objetivo la creación de una aplicación web y un videojuego para la empresa Verqor. La aplicación y el videojuego están destinados a enseñar a los clientes de Verqor las ventajas de sus servicios. El proyecto fue desarrollado utilizando HTML, CSS, React, NodeJS, MySQL y JavaScript. Se utilizo TailWind para el diseño responsivo y la creación de una interfaz de usuario atractiva. El videojuego fue diseñado para ser interactivo y educativo, permitiendo a los usuarios aprender sobre los servicios de Verqor de una manera divertida.
