Descripción del Proyecto

Este proyecto tuvo como objetivo la cración de una aplicación web para la Asociación Civil Dibujando Sonrisas, Dejando Huellas A.C. La aplicación esta dividad entre el área de administración dedicada a la gestión de los asocidados, padrinos, ahijados y donaciones, y el área publica donde se muestra unicamente información sobre la asociación. La aplicación fue desarrollada en PHP y MySQL, y se utilizó Bootstrap para el diseño responsivo. El proyecto fue un éxito y ha mejorado la gestión de la asociación civil.
