Descripción del Proyecto

Este proyecto tuvo como objetivo la creación de una aplicación para Android y una aplicación web para la Asociación Civil Todas Brillamos. La aplicación está destinada a la venta de productos de higiene personal y la administración de los mismos. El proyecto fue desarrollado utilizando HTML, CSS, React, NodeJS, MySQL y JavaScript. Se utilizó TailWind para el diseño responsivo y la creación de una interfaz de usuario atractiva. La aplicación web permite a los administradores gestionar los productos, pedidos y usuarios de manera eficiente. Los usuarios pueden navegar por el catálogo de productos, realizar pedidos y gestionar su cuenta de manera sencilla. La aplicación para Android ofrece una experiencia similar, permitiendo a los usuarios acceder a la tienda desde sus dispositivos móviles.
